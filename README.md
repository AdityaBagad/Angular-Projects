# Angular-Projects
This repository contains some basic Angular Applications.

## Prerequisites
1. Download and Install npm (Node Package Manager) : [Download](https://nodejs.org/en/)
2. Install CLI(Command Line Interface) : 
```
npm install -g @angular/cli
```
